# Курсовая работа: Количественные оценки универсальной аппроксимации ReLU-сетями

## Описание
В работе исследуется способность глубоких и широких ReLU-сетей аппроксимировать различные типы функций (гладкие, кусочно-линейные, смешанные). Проводится сравнение с сигмоидальными сетями. Эксперименты выполнены с усреднением по 5 запускам для статистической надёжности.

## Требования
- Python 3.9 или выше
- Библиотеки, указанные в `requirements.txt` (torch, numpy, matplotlib, scikit-learn)
